# Haven Blues
A game created in 4-5 months in Unity inspired by Harvest Moon and Zelda. 

# Interactables
- Basic inventory system
- Coin and Heart system
- Damage and Knockback system from enemies
- Scene Transitions
- Collectables
- Walking NPC's
- Enemy death counter
- Enemy Loot / Basic probability loot 
- Several scenes
- Breakable Pots

# Disclaimer 

All of the seen sprites are not mine and were taken from itch.io by their respective creators.
